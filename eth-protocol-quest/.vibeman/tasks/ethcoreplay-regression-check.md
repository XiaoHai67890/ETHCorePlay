# 回归检查任务（ETHCorePlay）

## 目标
在最近多轮迭代后做一次稳定性回归，重点排查白屏、跳转错误、交互失效和数据状态异常。

## 路由清单（必须检查）
- `/` 首页
- `/curriculum`
- `/progress`
- `/glossary`
- `/map`
- `/search`

## 检查项
1. 页面加载
- [ ] 无白屏
- [ ] 无致命 JS 报错

2. 关键交互
- [ ] 首页 CTA 正常跳转
- [ ] 课程章节展开/收起正常
- [ ] 章节测评可提交
- [ ] 术语图谱搜索/定位/固定节点可用
- [ ] 地图路径切换与图例过滤可用
- [ ] Command-K 可打开、搜索、跳转

3. 状态与数据
- [ ] 本地学习状态读取正常（无 undefined 崩溃）
- [ ] 总览页面成长阶段计算正常
- [ ] 云同步按钮点击不导致页面异常

4. 兼容性
- [ ] 桌面 Chrome（主）
- [ ] 移动端模拟（375x812）

## 验收标准
- [ ] 所有 P0/P1 功能可走通
- [ ] 无阻断级 bug（白屏/无法导航/功能不可用）
- [ ] 回归结论输出为“通过/阻断项列表”

- 回归结果：通过 / 不通过
- 阻断项：
